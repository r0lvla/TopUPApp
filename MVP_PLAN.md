### Ценовая модель — ступенчатая наценка

**Принцип:** дешёвые карты продаём с большей наценкой (компенсация усилий на мелкий чек), дорогие — с меньшей (конкурентная цена для крупных покупателей).

| Категория | Номиналы | Наценка | Целевая маржа |
|-----------|----------|---------|---------------|
| 🟢 **Дешёвые** | Малые номиналы | **50%** | ~30% |
| 🟡 **Средние** | Средние номиналы | **45%** | ~25% |
| 🔴 **Дорогие** | Крупные номиналы | **40%** | ~22% |

**Позиционирование:**
- Чуть дешевле раскрученных конкурентов (PayTool, GGSel — те берут 40-80%+)
- Чуть дороже ботов-полуфабрикатов (без гайдов, без поддержки)
- Премиум UX + интерактивный гайд смены региона = стоит своих денег

- KYC-lite: аккаунт Telegram >30 дней

#### 🔑 ИНСТРУКЦИЯ ПО СМЕНЕ РЕГИОНА — КЛЮЧЕВАЯ ФИЧА MVP

> **Это НЕ проблема — это УСЛУГА и конкурентное преимущество.**
> Без смены региона карты НЕ активируются. Apple упростила процедуру для россиян (апрель 2026).

**Что включаем в MVP:**

1. **Интерактивный гайд внутри Mini App:**
   - Пошаговая инструкция с анимацией
   - Выбор целевого региона (TR / US / KZ)
   - Кнопка «Показать адрес» — генерация реального адреса для выбранной страны
   - Для США: рекомендуемые штаты (Делавэр, Монтана, Орегон — нет налога)

2. **Предупреждение ПЕРЕД покупкой:**
   - «Этот код работает только на Apple ID с регионом [ТУРЦИЯ]»
   - Кнопка «Нужна помощь со сменой региона?» → гайд
   - Чекбокс «Я подтверждаю, что мой регион Apple ID = [ТУРЦИЯ]»

3. **Шаги гайда:**
   ```
   ШАГ 1: Подготовка
   • Баланс Apple ID = 0 (потратьте или запросите списание)
   • Отмените Apple Music (если есть)
   • Выйдите из Семейного доступа (если организатор)

   ШАГ 2: Смена региона
   • Настройки → [Имя] → «Медиаматериалы и покупки»
   • «Просмотр учётной записи» → «Страна/Регион» → «Изменить»
   • Выберите страну → Примите условия
   • Способ оплаты → «Нет» (None)
   • Адрес → скопируйте из нашего гайда (любой реальный адрес)
   • Совет: включите VPN страны перед сменой

   ШАГ 3: Активация карты
   • App Store → Аватар → «Погасить подарочную карту или код»
   • Введите 16-значный код → Баланс зачислен!

   ШАГ 4: Использование
   • Покупайте приложения, продлевайте подписки
   • iCloud+ продлится автоматически
   • Apple Music — переоформите в новом регионе
   ```

4. **Альтернатива — второй аккаунт:**
   - «Не хотите менять регион основного аккаунта? Создайте второй Apple ID для App Store»
   - Краткая инструкция по настройке

5. **Пост-покупка:**
   - Код + инструкция по активации → в чат Telegram
   - Кнопка «Проблемы с активацией?» → поддержка

#### Корзина и оплата
- Одно-товарная покупка (без корзины для MVP)
- Оплата: ЮKassa → СБП + банковские карты
- Через Telegram Payments API (sendInvoice + openInvoice)
- Нативная форма внутри Mini App

#### Доставка кода
- Мгновенная (<30 сек после подтверждения оплаты)
- Код отображается в Mini App
- Кнопка «Скопировать код»
- Отправка кода в чат Telegram (ботом) как подтверждение
- **Инструкция по активации для региона** (обязательно!)

#### Поддержка
- Кнопка «Поддержка» → переход в чат с @support_bot
- FAQ: как активировать, как сменить регион, возвраты
- Политика: возврат ТОЛЬКО до активации кода

#### Админка
- Простой веб-интерфейс (защищённый паролем)
- Функции:
  - Добавление/удаление номиналов
  - Просмотр заказов (статус, сумма, пользователь)
  - Статистика: кол-во заказов, выручка за день/неделю
  - Управление остатками (предупреждение при <5 шт)
  - Ручная выдача кода (fallback)
